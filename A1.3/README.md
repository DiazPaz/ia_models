# A1.3 Solución de problemas y selección de características

Este directorio contiene los archivos correspondientes a la actividad **A1.3** realizada para la clase de **Inteligencia Artificial 1**.

## Contenido del Folder
* [**Código Fuente:**](main.html) `main.ipynb` con el preprocesamiento y modelado.
* [**Reporte de Análisis:**](main.md) `main.md` con el detalle de la selección de características.
* [**Conjunto de Datos:**](calificaciones.csv) `calificaciones.csv`.

## Descripción General
Esta actividad se enfoca en predecir la calificación final (G3) de estudiantes de secundaria. Se implementa una regresión lineal múltiple utilizando **Lasso** para la selección automática de características, identificando que el desempeño en periodos previos (G2) es el predictor más fuerte.
