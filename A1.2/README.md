# A1.2 Regresión Lineal

Este directorio contiene los archivos correspondientes a la actividad **A1.2** realizada para la clase de **Inteligencia Artificial 1**.

## Contenido del Folder
* **Código Fuente:** [`main.ipynb`](main.ipynb) con la implementación en Python.
* **Reporte de Análisis:** [`main.md`](main.md) y [`main.pdf`](main.pdf) con la interpretación estadística.

## Descripción General
El objetivo es analizar la relación entre el nivel de felicidad y el GDP mediante regresión lineal simple. Posteriormente, se extiende a un modelo de regresión múltiple incorporando esperanza de vida, desempleo y escolaridad para evaluar qué factores estructurales impactan más en el bienestar social.
