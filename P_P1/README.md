# P1 Proyecto Final - Unidad 1

Este directorio contiene los archivos correspondientes al **Proyecto Final de la Unidad 1** de **Inteligencia Artificial 1**.

## Contenido del Folder
* **Código Fuente:** [`main.ipynb`](main.ipynb) y [`main.html`](main.html) con el flujo completo de ciencia de datos.
* **Reporte de Análisis:** [`main.pdf`](main.pdf) que documenta todo el proyecto socioeconómico.
* **Conjunto de Datos:** [`base_pobreza.csv`](base_pobreza.csv)  (datos ENIGH 2024) y [`diccionario_datos.csv`](diccionario_datos.csv) .

## Descripción General
El proyecto estima el ingreso trimestral per cápita en Nuevo León a partir de las características de la vivienda y servicios básicos. Se contrastan modelos **OLS** y **Ridge (Polinomial)** con los umbrales de pobreza del **CONEVAL** para identificar factores críticos de vulnerabilidad económica.
